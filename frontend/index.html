API endpoints

url method response
/table GET Returns all tables in an array
/table PUT Creates a new table and returns it
/table/:tableId GET Returns the data set from the database when found by tableId.
/table/:tableId PUT Sets occupied to true and occupiedUntil to the moment when the function is called of the table matching to the given parameter.
/table/occupy/:tableId GET Finds the table with the given Id and set it occupied. It will be reset after 30 minutes by the server.

Sockets

name description
table::occupied::true Table was set to occupied.
table::occupied::false Table was set to free.
